常见问题

FAQ

下载链接打不开怎么办？ ▼
已添加国内加速代理（ghfast.top）。如果仍然有问题：
1. 刷新页面重试
2. 切换网络（热点/不同的 Wi-Fi）
3. 直接访问 GitHub Releases 页面手动下载
工具安全吗？会泄露我的聊天记录吗？ ▼
完全安全。两款工具都是开源项目，源代码公开可查。
关键：它们完全离线运行。你的聊天数据不需要上传到任何服务器，解密和导出全部在你自己电脑上完成。第三方无法获取你的任何数据。
下载时浏览器报"病毒"或"不安全"怎么办？ ▼
这是误报，不是真病毒。
原因是这两个工具是个人开发者做的开源项目，没有买微软的代码签名证书（每年几千块）。Windows Defender、Chrome、Edge 会对"未签名"的 .exe 弹警告，但这不是检测到病毒，只是"发布者未知"。

验证安全的方法：
1. 两个项目在 GitHub 上代码完全公开，可以自己查看
2. wx_key 累计 15000+ 次下载，零安全投诉
3. 如果还是不放心，下载后用 VirusTotal 在线扫描验证
解密后文件多大？怎么用？ ▼
取决于你的聊天记录量，一般从几 MB 到几十 MB 不等。导出的 TXT 文件可以用记事本打开，也可以直接拖拽到 AI 平台（豆包、Kimi 等都支持上传文件）。
微信更新后工具还能用吗？ ▼
WeChatDataAnalysis v1.9.1（最新版）已适配最新微信版本。如果微信更新后工具不可用，请关注对应 GitHub 项目的 Release 页面，作者通常会很快适配。
手机能下载吗？没有电脑怎么办？ ▼
这两个工具必须在 Windows 电脑上运行，手机上打不开。

但你可以提前准备：
1. 用手机点「加速下载」把文件存到手机里
2. 用数据线或微信文件传输助手传到电脑上
3. 到了电脑上直接双击安装就行

或者收藏这个页面，等有 Windows 电脑了再弄。
为什么下载文件这么大（99MB）？ ▼
WeChatDataAnalysis 包含了完整的运行环境和所有依赖，安装后即可直接使用，无需额外配置。99MB 对于现代网络来说下载约 1-3 分钟。

提取密钥

大约 3 分钟，获取微信数据库的解密密钥

1
打开微信 PC 版并登录
确保微信电脑端已经登录，并且可以看到你的聊天记录列表。
2
右键 wx_key.exe → 以管理员身份运行
解压下载的压缩包，右键点击 wx_key.exe，选择"以管理员身份运行"。
3
点击"开始提取密钥"按钮
在 wx_key 界面中，点击醒目的绿色按钮，等待几秒钟。
4
复制数据库密钥
工具会显示一串密钥字符串，点击复制按钮将密钥保存备用。

第三步

解密导出

大约 5 分钟，将加密的数据库导出为可读的文本文件

1
打开 WeChatDataAnalysis，点击"检测本机微信数据"

2
选择数据库文件（MSG0.db）
检测完成后，选择要解密的数据库文件。
3
粘贴数据库密钥
将之前复制的密钥粘贴到密钥输入框中。
4
点击解密

5
导出为 TXT 文件
解密完成后，选择导出格式为 TXT，保存到本地即可。

FAQ

常见问题

下载链接打不开怎么办？ ▼
如果遇到下载缓慢或链接打不开，我们已经添加了国内加速代理（ghfast.top）。如果仍然有问题，请尝试刷新页面或更换网络环境。也可以尝试直接访问 GitHub Releases 页面手动下载。
这些工具安全吗？会不会泄露隐私？ ▼
完全安全。这两款工具都是开源项目，代码公开可查。它们完全离线运行，数据不需要上传到任何服务器。你的聊天记录只留在你的电脑上。
解密后的文件是什么格式？ ▼
解密后可以导出为 TXT 纯文本格式，也可以用 WeChatDataAnalysis 的内置查看器直接浏览。你可以复制粘贴到任何 AI 工具中进行分析。
支持 Mac 或 Linux 吗？ ▼
目前 wx_key 和 WeChatDataAnalysis 主要支持 Windows 系统。Mac 用户可以尝试使用 Wine 运行，或寻找其他跨平台替代工具。
微信版本更新后工具还能用吗？ ▼
WeChatDataAnalysis 最新版 v1.9.1 已适配最新微信版本。如果遇到微信更新后工具不可用，请关注 GitHub 项目的更新，通常作者会很快适配新版。

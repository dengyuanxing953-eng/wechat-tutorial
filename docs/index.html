微信聊天记录导出
到 AI 分析

适用场景：把微信群、好友聊天记录导出成 TXT，再让 AI（ChatGPT / Claude / 豆包等）帮你总结、分析、整理

预计耗时 10-15 分钟
难度等级 ★☆☆☆☆ 入门
适用平台 Windows 10/11
版本 v2.1
01
前期准备 PREPARATION & REQUIREMENTS

在开始操作之前，请确认您已具备以下基本条件。

项目 要求 状态检查
操作系统 Windows 10 或 Windows 11 设置 → 系统 → 关于
电脑微信 已安装并登录，聊天记录在电脑上 打开微信能看到历史聊天
网络环境 能正常访问互联网 浏览器能打开网页

本次操作需要用到两个工具：

wx_key

密钥提取

微信聊天记录是加密的，这个软件帮你拿到解密所需的"钥匙"（数据库密钥）

WeChatDataAnalysis

解密导出

用获取到的密钥打开微信数据库，将聊天记录导出为可读的文本文件

02
工具下载 SOFTWARE DOWNLOAD
2.1 下载 wx_key
1

打开 GitHub 发布页

访问 github.com/ycccccccy/wx_key/releases

2

下载压缩包

找到 wx_key-windows-v2.1.8.zip，点击下载（约 50MB）

2.2 下载 WeChatDataAnalysis
1

打开 GitHub 发布页

访问 github.com/LifeArchiveProject/WeChatDataAnalysis/releases

2

下载安装程序

找到 WeChatDataAnalysis-1.8.1-Setup.exe，点击下载（约 200MB）

!
如果 GitHub 无法正常访问，可通过百度搜索关键词 "wx_key 下载" 或 "WeChatDataAnalysis 下载" 找到其他下载源。
03
提取数据库密钥 EXTRACT DATABASE KEY
3.1 解压 wx_key

右键点击下载好的 wx_key-windows-v2.1.8.zip → 选择 "解压到当前文件夹" → 获得 wx_key.exe

3.2 运行 wx_key
i
运行前请确保电脑微信已登录并保持打开状态。右键点击 wx_key.exe → 选择"以管理员身份运行" → 在弹出的 UAC 提示中点击"是"。
3.3 提取密钥
1

点击"开始提取密钥"

软件会自动检测微信的数据库路径并提取密钥

2

等待提取完成

通常几秒钟即可完成，界面会显示一串 64 位的十六进制字符

3

复制密钥

选中并复制这串密钥

i
这串密钥长这样（示例）：a1b2c3d4e5f6...（共 64 位十六进制字符），请确保复制完整。
3.4 保存密钥

在桌面新建一个名为 微信密钥.txt 的文本文件，将密钥粘贴进去并保存。

⚠
重要安全提示：密钥是你微信数据的钥匙，请妥善保管，不要发给陌生人。使用完成后建议删除此文件。
04
解密并导出聊天记录 DECRYPT & EXPORT CHAT RECORDS
4.1 安装 WeChatDataAnalysis

双击下载的 .exe 安装程序 → 如出现 Windows 安全提示，点击 "更多信息" → "仍要运行" → 按向导完成安装。

4.2 打开软件并检测微信

启动 WeChatDataAnalysis → 点击 "开始检测"，软件会自动扫描微信的数据库路径。

4.3 输入密钥并解密
1

粘贴密钥

将之前保存的 64 位密钥粘贴到软件的"解密密钥"输入框中

2

确认数据库路径

检查路径是否正确指向微信的数据库文件夹

3

开始解密

点击 "开始解密"，等待 1-5 分钟完成解密

4.4 找到目标聊天并导出

解密完成后，在左侧列表中搜索你想导出的聊天（群名或好友名），点击选中，然后按以下设置进行导出：

设置项 推荐值 说明
导出范围 当前会话 只导出选中的聊天
导出格式 TXT 纯文本格式，AI 可直接读取
文本 ✅ 勾选 文字消息（核心内容）
聊天记录 ✅ 勾选 合并转发的消息
链接 ✅ 勾选 分享的网页链接
引用 ✅ 勾选 引用回复的消息
系统 ✅ 勾选 系统提示消息
图片 ❌ 不勾选 AI 无法直接分析图片
表情 ❌ 不勾选 表情包占空间
视频 ❌ 不勾选 文件体积过大
语音 ❌ 不勾选 AI 无法直接分析语音
文件 ❌ 不勾选 文件类型不统一
导出目录 桌面 方便查找导出结果

设置完成后点击"导出"按钮。

05
定位导出文件 LOCATE EXPORTED FILES

导出完成后，在桌面找到 conversations 文件夹，目录结构如下：

📁 conversations/ ├── 📁 0001_群聊名称_xxx/ │ └── messages.txt ← 就是这个文件 ├── 📁 0002_好友名称_xxx/ │ └── messages.txt └── ...
✓
打开 messages.txt，里面就是完整的聊天记录文本，可以直接发送给 AI 进行分析。
06
AI 分析操作 AI ANALYSIS WORKFLOW
6.1 使用 ChatGPT 分析
1

打开 ChatGPT

访问 chat.openai.com，登录账号

2

新建对话并上传文件

点击 "New Chat"，使用附件功能上传 messages.txt

3

输入分析提示词

参考以下 Prompt 模板发送：

请分析这段聊天记录，总结以下内容：
1. 对方的性格特点
2. 我们的沟通模式
3. 关键事件和时间线
4. 对方对我的态度变化
6.2 使用 Claude 分析

操作流程与 ChatGPT 完全一致：访问 claude.ai → 新建对话 → 上传 messages.txt → 输入同样的分析提示词即可。

⚠
隐私保护建议：
• ChatGPT：Settings → Data controls → 关闭 "Improve the model for everyone"
• Claude：Settings → Privacy → 关闭 "Help improve Claude"
这样可以防止你的聊天数据被用于模型训练。
07
常见问题 FREQUENTLY ASKED QUESTIONS

请按以下步骤排查：① 确认微信已登录并保持打开状态；② 以管理员身份运行 wx_key；③ 如果仍然无法提取，可能是微信版本更新导致工具暂时失效，建议关注工具的 GitHub 页面等待更新。

检查软件检测到的数据库路径是否正确，路径应指向微信的 db_storage 文件夹。如果路径不正确，可手动指定微信数据库的默认存储位置（通常在用户目录下）。

可以采取两种方案：① 在导出时按时间范围分批导出（如按月导出）；② 导出后使用文本编辑器手动将文件拆分为多个小文件，每次上传一个进行分析。

在第 4.4 步中，在聊天列表中搜索该好友的名字，选中对应会话，导出范围选择"当前会话"即可。

可尝试以下方法：① 使用网络代理工具访问；② 在百度搜索相关工具名称，寻找国内镜像或网盘下载源；③ 在 GitHub 相关技术社区（如 V2EX、知乎）搜索最新的下载链接。

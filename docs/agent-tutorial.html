STEP 1 / 6

下载 Cursor

Cursor 是一个带 AI 的代码编辑器，免费够用。
打开 cursor.com，点 Download，安装。

Mac 双击 .dmg 拖进应用程序；
Windows 双击 .exe 一路下一步。

💡 装完打开一次，让它走完初始化。看到欢迎界面就关掉，不用注册。

STEP 3 / 6

拿一把钥匙

去对应的平台注册，创建 API Key：

打开 platform.deepseek.com，手机号注册。

进「API Keys」→ 点「创建 API Key」→ 复制。

⚠️ 钥匙只显示一次。复制后立刻粘到备忘录。然后去「充值」充 5 块钱。

打开 dashscope.aliyun.com，用支付宝/淘宝登录。

进「API-KEY 管理」→ 点「创建」→ 复制。

💡 千问有免费额度，新用户送 100 万 tokens，不用充值就能开始。

打开 platform.moonshot.cn，手机号注册。

进「API Key 管理」→ 点「创建」→ 复制。

💡 新用户有免费额度，先体验再充值。

打开 platform.mimo.xiaomi.com，手机号注册。

进「API Keys」→ 点「创建」→ 复制。

💡 MiMo 推理能力强，代码任务表现好。新用户有免费额度。

👆 先在上一步选一个服务商，这里才会显示对应的注册指引。

STEP 4 / 6

把钥匙插进 Cursor

打开 Cursor，按步骤操作：

① 点右上角齿轮 ⚙️ → Settings
② 左侧点 Models
③ 找到 OpenAI API Key 那一栏，点 Configure
④ 粘贴你的 API Key，点 Verify

⑤ 还需要填 API Base URL：

https://api.deepseek.com/v1
💡 为什么用 OpenAI 格式？国产 API 都兼容 OpenAI 接口，Cursor 原生支持，不用装插件。

STEP 5 / 6

给你的 AI 助手一份说明书

这一步让 Cursor 变成你的专属助手。
复制下面的规则，粘进 Cursor：

① Cursor 左下角齿轮 ⚙️ → Rules
② 点 Add new rule
③ 粘贴下面的内容，保存

你是一个友好的 AI 编程助手。 ## 你的职责 - 帮助零基础用户学习编程和使用 AI 工具 - 回答用中文，简洁明了，不用专业术语 - 每次操作前先解释"这是什么"和"为什么要这样做" - 遇到报错，先翻译成人话，再给解决方案 ## 当用户说"帮我装 Claude Code" 1. 先确认用户的系统（Windows / Mac） 2. Windows 用户：用 winget install Anthropic.ClaudeCode 安装 3. Mac 用户：用 brew install --cask claude-code@latest 安装 4. 装完后帮用户验证：运行 claude --version 5. 告诉用户怎么启动：终端输入 claude 回车 ## 代码风格 - 写完代码要解释每一行在干什么 - 优先用最简单的方案 - 出错不要慌，一步步排查

STEP 6 / 6

让 AI 帮你装 Claude Code

在 Cursor 里打开 Composer（快捷键 Ctrl+I 或 Cmd+I），
把下面这句话粘进去，回车：

帮我安装 Claude Code，我是 Windows/Mac 用户，装完帮我验证一下

AI 会自动帮你执行安装命令。
过程中如果弹出权限提示，点允许。

💡 装完之后，打开终端（Windows: PowerShell，Mac: 终端），输入 claude 回车，看到欢迎界面就成功了！
